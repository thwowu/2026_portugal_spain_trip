# 塞維爾延伸行程（詳情）

## cordoba | Córdoba（一日：清真寺主教座堂/老城）
### summary
- 適合：Seville 多住 +1 天，或 Semana Santa 市中心太擠想「逃離人潮」
- 亮點：老城集中、步行好走、非常值得的一日遊

### how
- 火車：Renfe（AVE/高速）Sevilla ↔ Córdoba（先看班次再決定出發時間）
- 票務：建議先在 Renfe 查班次/買票（旺季更需要）

### route
- 一日動線（不趕版）：上午早班到 Córdoba → 老城散步 → 下午回 Sevilla

### time
- 單程火車約 45 分鐘（以 Renfe 班次為準）+ 市區步行為主

### backup
- Semana Santa 期間：Sevilla 市中心封路擁擠時，Córdoba 會更舒服（但同樣可能人多，早出門）

### sources
- Renfe（Sevilla–Córdoba route）：https://www.renfe.com/es/en/experiencias/rutas/sevilla-cordoba
- 交通與走法（經驗文）：https://mapandcamera.com/seville-to-cordoba-train/

## cadiz | Cádiz（看海放鬆：海邊城市散步）
### summary
- 適合：想加一個「看海」的備案日（不想一直逛宮殿教堂）
- 亮點：海邊散步＋放空日

### images
- {{gallery:Cádiz|https://commons.wikimedia.org/wiki/Special:FilePath/Playa_de_la_Caleta,_C%C3%A1diz,_Espa%C3%B1a,_2015-12-08,_DD_53.JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz_playa_caleta.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz._Puesta_de_sol_en_Mayo.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Exterior_Catedral_de_C%C3%A1diz_1.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Kathedrale_in_C%C3%A1diz.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Playa_de_la_Caleta,_C%C3%A1diz,_Espa%C3%B1a,_2015-12-08,_DD_53.JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz_playa_caleta.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz._Puesta_de_sol_en_Mayo.jpg?width=1800}}

### how
- 火車：Sevilla ↔ Cádiz（以 Trainline/Renfe 查班次）

### route
- 一日動線：到 Cádiz → 海邊散步/吃海鮮 → 傍晚回 Sevilla

### time
- 單程約 1h40 左右（以班次為準）

### sources
- Trainline（Cádiz ↔ Seville 時刻/購票入口）：https://www.thetrainline.com/en/train-times/cadiz-to-seville
- Trainline（Seville → Cádiz 時刻）：https://www.thetrainline.com/en/train-times/seville-to-cadiz
