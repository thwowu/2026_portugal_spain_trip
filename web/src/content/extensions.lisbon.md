# 里斯本延伸行程（詳情）

## obidos-nazare | Óbidos + Nazaré（中世紀小鎮 + 海岸懸崖）
### summary
- 適合：多住 +1 天 / 把某天當備案（下雨或前面太累）
- 亮點：Óbidos 走城牆＋白牆巷弄、Nazaré 看懸崖視角（浪況看運氣）

### when
- 最適合放：里斯本行程最後（體力比較回來了），或當作「前面某天太累/下雨」的補救日
- 復活節週：公車/路上人會多，盡量早出門、不要把回程壓到太晚

### how
- 交通方式（不自駕最實際通常是巴士）
  - Lisboa → Óbidos：長途巴士（常見從 Campo Grande / Sete Rios 一帶出發；以官方查詢為準）
  - Óbidos ↔ Nazaré：通常需要接駁/轉乘（若不想處理轉乘，建議直接選「一日團/包車」）
  - Nazaré → Lisboa：長途巴士回城
- 訂票/查班次（官方入口）
  - Rede Expressos：用官方「Horários e bilhetes」搜尋站點與班次

### route
- 版本 A（最不燒腦）：早到 Óbidos（走城牆＋巷弄）→ 午後 Nazaré（Sítio 懸崖區視角）→ 傍晚回里斯本
- 版本 B（怕拉車）：只做 Óbidos 半日 + 里斯本晚餐（把 Nazaré 留給更想看海時）

### time
- 一日遊：早出晚回
- Óbidos 約 2–3 小時 + Nazaré 約 1–2 小時（其餘時間多在交通）

### cost
- 交通票價會因班次/購票時間變動：以 Rede Expressos 搜尋結果為準
- 如果你選一日團：用「省轉乘風險」換時間/金錢（尤其復活節週很值得）

### backup
- Plan B（下雨/風太大）：Nazaré 不追海邊，改成「只去 Óbidos」或把 Nazaré 當作遠看/喝咖啡點
- Plan B（班次對不上）：直接換成「Cascais」這種低風險近郊（你本來就會走）

### sources
- VisitPortugal（Óbidos）：https://www.visitportugal.com/es/content/obidos
- Rick Steves 社群討論（交通/安排經驗）：https://community.ricksteves.com/travel-forum/portugal/nazare-obidos-coimbra-day-tripping-out-of-lisbon
- Rede Expressos（官方查班次/買票入口）：https://rede-expressos.pt/pt/horarios-bilhetes

## evora | Évora（UNESCO 古城：羅馬神殿/老城）
### summary
- 適合：想要「不是海岸線」的備案日，或想補葡萄牙古城感
- 亮點：老城步行、羅馬遺跡、教堂/廣場

### images
- {{gallery:Évora|https://commons.wikimedia.org/wiki/Special:FilePath/Templo-diana-evora-portugal.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Templo_Romano_de_Diana_-_%C3%89vora_-_Portugal_(296376498).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Ancient_Roman_Temple,_%C3%89vora_-_Apr_2011.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Templo-diana-evora-portugal.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Templo_Romano_de_Diana_-_%C3%89vora_-_Portugal_(296376498).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Ancient_Roman_Temple,_%C3%89vora_-_Apr_2011.jpg?width=1800}}

### how
- 交通方式（兩種都可行，取決於你住哪/想要多彈性）
  - 火車：CP 查 Lisboa（Oriente/Sete Rios 等）→ Évora（班次以官網為準）
  - 巴士：Rede Expressos（同樣用官方查班次/買票入口）
- 市內移動：Évora 老城主要靠走路即可（穿好走鞋）

### route
- 一日走法（最典型）：老城散步 →（想看就看）教堂/廣場/羅馬遺跡 → 早點回里斯本

### time
- 一日遊：早出晚回（老城可用步行完成）

### backup
- Plan B（下雨）：照樣能走（多進教堂/博物館），只要把戶外散步縮短

### sources
- VisitPortugal（Évora World Heritage）：https://www.visitportugal.com/en/destinos/alentejo/73787
- CP（官方查火車班次）：https://www.cp.pt/passageiros/pt/consultar-horarios
- Rede Expressos（官方查班次/買票入口）：https://rede-expressos.pt/pt/horarios-bilhetes

## cascais | Cascais（近郊海岸線：你行程 Day 3）
### summary
- 適合：不想拉太遠、需要「低風險放鬆日」
- 亮點：海岸線散步＋小鎮氛圍

### images
- {{gallery:Cascais|https://commons.wikimedia.org/wiki/Special:FilePath/Ba%C3%ADa_de_Cascais.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Boca_do_Inferno_-_Cascais_-_Portugal_(269119308).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Bocadoinferno.jpg?width=1800|https://images.pexels.com/photos/24377407/pexels-photo-24377407.jpeg?auto=compress&cs=tinysrgb&w=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Ba%C3%ADa_de_Cascais.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Boca_do_Inferno_-_Cascais_-_Portugal_(269119308).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Bocadoinferno.jpg?width=1800}}

### how
- 火車：Lisboa（Cais do Sodré）↔ Cascais（CP 市區線；班次密集，旺季提早去站內即可）

### route
- 半日～一日：Cascais 小鎮 → Boca do Inferno → 海邊散步 → 回里斯本吃晚餐

### sources
- CP（Cascais Line / Stations）：https://www.cp.pt/passageiros/en/train-times/Stations/cascais-station
