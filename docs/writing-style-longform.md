# Writing Style – Longform / Personal Experience (非 SSOT)

這份文件收納「需要第一人稱、需要親身經驗才成立」的長文技巧。

## 使用前提（很重要）

- 只寫你真的經歷過的細節：看見的人、聽到的音樂、對話、情緒轉折，都不要用網路資料硬湊。
- SSOT 文件不要引用這裡的模板：若內容要進 `docs/src/content/*`（SSOT），請回到 `docs/writing-style-guidelines.md` 的規範。

## 可用風格模組（長文才有效）

### 1) 長文共用骨架（預設骨架，不是硬規定）

註：段落數只是「好用的預設」，後面的子型可以自由加段、拆段或改順序；重點是讓讀者跟得上你的推進與情緒落點。

- 開場（1 段）：時間/距離/限制（今天只有半天、我只是過境、走了七分鐘到海邊…）＋一句動機。
- 中段（2–5 段）：用「動作 → 感受」推進，每段只抓 2–3 個可感受細節（物件/聲音/人群）。
- 決策（可選 1 段）：把推薦寫成「我當下怎麼選」，或乾脆寫成「我為什麼決定停下來/不去/改走另一條路」（不要命令讀者）。
- 收束（1–2 句 + 1 動作）：一句結論 + 一個小動作落地（結帳/投小費/回房間/出門吃晚餐）。

### 2) 子型 A：生活切片/沉浸散文（慢時間、感官與群像）

- 數字落地：幾分鐘、幾小時、幾段路。
- 群像：用 2–3 種人把場景立起來（不要寫「很多人」）。
- 購物/選品段落的小招（很適合套用）：
  - 克制宣言當鉤子：先寫「我現在先不要買，因為真正要買的是 ___」→ 讀者會等你「抵達那家店」。
  - 品牌翻譯 + 一句使命：把名字翻成中文/概念，再用 1–2 句講清楚它的選品哲學（不要寫成百科）。
  - 空間就是賣點：用 2–3 個建築/陳列細節讓人看到「光看就開心」的理由（拱門、挑高、貨架、磁磚、老工廠改建…）。
  - 內心清單 = 決策：用幾句引號把腦內購物清單演出來（買來送人/買來自用/這次補什麼），比「推薦必買」更像真的。
  - 走位式逛店：用「繞一圈又一圈、每圈籃子變重」寫節奏（不用把每個商品都點名）。
  - 結帳線索做轉場：店員小卡/折價券/新分店資訊 → 自然把下一段行程引出來（不用硬寫「然後我去了…」）。
- 街邊小吃/小店短文的小招（很適合套用）：
  - 城市摩擦當引子：先寫一個「身體感」的摩擦（坡、熱、風、餓）→ 再寫你為什麼仍急著走（目標店在前面）。
  - 先把食物定義清楚：用 1 段交代「這是什麼」＋ 2–4 個關鍵材料/調味（就能立刻讓人想像味道）。
  - 「到處都有」反而是推薦理由：因為隨處可見，所以名店更值得（用這個轉折替「為什麼是這家」背書）。
  - 香氣預告：靠近時先寫氣味/熱氣作為預告，讀者會跟著你加速。
  - 小店的限制寫清楚：沒座位、只能外帶/站著吃，會讓畫面更真，也幫讀者做心理準備。
  - 料理台一鏡到底：平底鍋白煙/咕嚕湯汁/老闆手勢 → 不用多，抓 2–3 個就夠。
  - 價格是錨點：用一個小數字把「日常感」落地（例：€3 這種）。
  - 第一口只要一句：用一個短感嘆（噢！）＋ 2–3 個味覺詞（濃郁/不柴/微嗆/嚼勁）就能成立。
  - 用人群當背書：旅人＋熟客的混合，比「很有名」更可信。
  - 收尾用回訪宣言：不用總結大道理，一句「離開前要再來」就很有記憶點。

- 餐廳/宵夜麵店（夜裡的小店，空間裝置當賣點，很適合套用）：
  - 用反問開場：如果夜裡巧遇這樣一間店，你會推門進去嗎？（立刻把「神祕感」變成動力。）
  - 把「不在預期的位置」寫出來：立吞店通常在車站附近，但它偏偏藏在河邊巷弄（用這個反差替它的存在合理化）。
  - 入口的資訊缺口：黃光、玻璃門、看不見招牌、看不懂裡面賣什麼（讓讀者跟著你遲疑）。
  - 你的身分轉折：你不是路過，而是做功課有備而來 → 推門進去（用一句話完成轉折即可）。
  - 空間規則 = 故事核心：桌子不是桌子（陶藝作品、傾斜的黑色方塊）→「怎麼放碗」變成小挑戰，讀者會跟著你笑。
  - 食物段落只要一個骨架：麵（每天現做）＋湯（昆布/鰹魚等出汁）＋配料（1–2 個）→ 收在一句「冬夜裡被熱湯救回來」。
  - 把它放回行程：比起正餐更像宵夜；如果是泡完錢湯後，幸福感會被放大（這是行程安排的理由）。
  - 外頭的人當鏡子：路人/外國旅人從玻璃外看見你們「站著吃麵」的困惑神情（用 1–2 句就能讓畫面活起來）。
  - 用一個微小的收尾細節：離開時才看到門上的小店名/標記（比大道理更有餘韻）。

- 午餐選店：咖啡店今日定食（很適合套用）：
  - 開場把午餐定義成「行程錨點」：午餐不是補能量而已，而是讓你把上午的熱和亂放下來，重新把下午排好。
  - 先說你的固定解法：一份今日定食＋一杯冰咖啡（讀者會立刻知道你怎麼選）。
  - 用城市特性當背書：京都這種咖啡店密度高的城市，「推門進去就有午餐」會變成一種可靠的幸福。
  - 不要直接推名店，先寫「你怎麼撲空」：錯過午餐時間、碰上店休、一直想吃但沒吃到 → 這次算準時間才成功（張力就出來了）。
  - 入口/動線做情緒鋪陳：從外頭看不出所以然 → 爬木樓梯上二樓 → 視野突然打開（這一段很像電影轉場）。
  - 空間三件套就夠：木地板、植物、漫畫/書架（用 2–3 個物件讓讀者坐下來）。
  - 等待時給一個小動作：去書架挑一本漫畫翻著等（用動作把「舒服」演出來）。
  - 餐點寫「份量剛好」：好吃之外，重點是吃完不會太飽，能把午後留給甜點/散步（這是選店理由）。
  - 收尾用人物比喻：把店比成一個可靠的朋友（不華麗、不打卡，但你會信賴它）。

- 食物介紹：點餐選擇題（漢堡排/定食很適合）：
  - 先把「選擇題」點出來：你不是在研究餐廳，而是在解一題：醬汁選哪個、份量選多少。
  - 用等待做開場張力：座位少、先在外面等 10–20 分鐘 → 入座吧台（讀者會立刻進到現場）。
  - 把菜單資訊寫成對照：`黑醬/白醬`、`150g/300g`、定食附什麼、飯能不能續（讀者會跟著你「天人交戰」）。
  - 高潮要押在「上桌的瞬間」：鐵板滋滋作響、氣味先到、再看到半熟蛋與小旗子（用 2–3 個畫面就夠）。
  - 切開半熟蛋當爆點：蛋黃流出那一下，等於替整段午餐下了句點（這種瞬間很值得寫）。
  - 飲料/酒做收束：如果你加點了一杯酒，把它寫成「替一個人旅行的午餐補上句點」的儀式感，而不是炫耀。

- 食物介紹：名物主軸＋變化款＋配角很強（漢堡排名店很適合）：
  - 先把主軸說清楚：這家店的名物就是漢堡排，其他都是「從主軸長出來的變化」。
  - 列 3–5 個變化款就夠：每日限定、經典醬、加炸蝦、加可樂餅…（不要把菜單全抄，抓能代表的幾個）。
  - 把配角寫成第二主角：米飯的產地/品種、玄米、免費續碗（這些會直接影響「值不值得專程」）。
  - 味覺形容要短、押在一句：肉汁/軟嫩/醬香，不要連用三個形容詞；下一句用「我為什麼滿足」收掉。
  - 用候位畫面做驗證：離店時看到外頭的人潮，反過來替你的推薦背書（也順便把「要開門前來」這種策略落地）。

- 料理研究/菜色入門（湯咖哩這種很適合）：
  - 先用對比把菜定義清楚：它和你以為的「咖哩」哪裡不一樣（濃稠淋飯 vs 香料湯高湯）。
  - 給出吃法的一句動作：舀一口飯 → 浸到湯裡 → 一口吃掉（讀者立刻能想像口感）。
  - 定番三件套：大量蔬菜＋主角肉＋一碗香氣很滿的湯（用「結構」而不是品牌名在說服）。
  - 主角偏好可以很直白：你偏愛雞腿/培根/加點串燒都行，但要寫成「我為什麼選它」（口感、風味、飽足感）。
  - 視覺先到位：色彩斑斕的蔬菜、帶骨肉的存在感、半顆蛋的點綴（抓 2–3 個就停）。
  - 最後回到儀式感：每次來都會安排一餐，因為它像城市限定的誠意（收在一句就好）。
- 奇遇型古物店/挖寶短文的小招（很適合套用）：
  - 對話鉤子要夠怪：一句突如其來的指令/邀請（去餵貓、來吃點心、隔壁還有一間）會讓讀者立刻進場。
  - 巧遇 → 立刻改行程：把推薦寫成「我真的照做了」（買票/搭車/走進商店街），旅途會突然有推進力。
  - 店主是一個角色：不用介紹身分背景，抓 1–2 個動作（把店開著去買菜、請你自己逛）就能立起來。
  - 小任務讓互動落地：餵貓/喝湯/幫忙拿東西這種微任務，會把「人情味」寫得像真的。
  - 空間反轉：店裡已經很滿了 → 後院還有 → 對街還有（用一個「嘿，這裡還有喔！」當轉折）。
  - 戰利品當記憶錨點：最後只留 1–2 件物件＋金額，讓回程的藍天/海風把情緒收回來。
- 住宿/旅館短文的小招（很適合套用）：
  - 差旅/匆匆來回當開場：先交代「原本不抱期待」→ 後面「意外成為心頭好」才會更有說服力。
  - 品牌體系用一段講完：如果是連鎖/系列旅館，用「等級梯」快速定位（例如 1–7 代表價位/服務），立刻讓讀者知道你住的是哪一種。
  - 用空間結構寫房間：把房間寫成動線：進門看見什麼 → 下層能做什麼 → 上層能做什麼 → 動線如何把小房間變好用（例如挑高/樓中樓/榻榻米）。
  - 找一個「設計主題」當記憶點：例如把浴室做成澡堂意象、用磁磚與木盆把地方氣質塞進房間。
  - 活動是加分，不是硬推：把旅館的在地導覽/夜間活動寫成「如果你想多一層體驗」的選項。
  - 最後回到你為什麼喜歡：不只說漂亮/方便，而是說它滿足了你哪個深層偏好（童年想要的房型、大小剛好能安放疲憊、太大反而會胡思亂想）。
  - 一句回訪結論就好：收在「下次還會住嗎？YES」這種乾脆句，比長總結更有力。

- 同品牌/分館重逢的旅館文（很適合套用）：
  - 先放一段「多年以前」：一間旅館在你人生某個階段留下了記憶（第一次獨旅、第一次睡得很好）。不要寫大道理，只寫 2–3 個物件細節（床頭燈、桌椅材質、房間比例）。
  - 再放一段「突然得知分館」：你在快閃/臨時行程前得知它在別的城市開了新館 → 立刻改住宿（這是推進力）。
  - 先講結論但要留鉤子：像「我真的愛瘋」這種一句結論可以先放，但下一段要立刻回答「它到底哪裡讓你愛」。
  - 把公共空間寫成藝廊：入口畫作、接待區像展覽、定期換檔、落地窗外的港景/夕陽（用 2–3 個鏡頭就好）。
  - 把房間寫成「你怎麼被照顧」：狹長挑高、陽台景、床頭兩盞燈的光線、咖啡機/藍牙喇叭這些小物件，讓「舒適」變具體。
  - 清晨段落用來收情緒：拉開窗簾看海港、船慢慢經過、再賴一下床（這段是長文的禮物）。
  - 早餐當高潮：品項很多不是重點，重點是「你只剩 30 分鐘也要吃」＋「景觀座位」＋「我好險有加訂」。
  - 收尾回扣時間軸：十多年前的你 vs 現在的你；同一個品牌仍能讓你快樂（用一段感謝收束即可）。
  - 最後用腳註落地：步行距離、到站時間、叫車方式、到機場車程（像你文末那樣，讀者會真的拿去用）。

### 3) 子型 B：行程推進型（一日/過境/待辦清單）

- 主題線：用一個意象貫穿（運河/坡/光/海風）。
- 節奏：重量級 1 個＋驚喜型 1 個＋甜點停靠 1 個。
- 尾聲：清單完成後「反而更慢」的收尾。

- 度假村/全待在園區裡的旅行（很適合套用）：
  - 對話開場：朋友一句話把你推向新玩法（你應該試試看都待在度假村裡）。
  - 先承認自己的舊習慣：平常很忙、很愛跑點/取材 → 才能寫出「度假」的反差。
  - 以度假村為圓心：除了半日出門，其餘都在園區內（把「不出門」寫成主題，不用覺得心虛）。
  - 用重複日常堆出慢時間：晨間活動 → 早餐 → 泳池/發呆亭 → 午睡/閱讀 → 夕陽 → 晚餐（重複本身就是度假的韻律）。
  - 具體但不碎：挑 2–3 個固定場景（泳池、水道、發呆亭/露台）反覆出現即可。
  - 最後把它變成一個新選項：寫「我在心裡植入新選項」比「這很棒」更像作者。

- 跨城列車/高鐵訂票與搭乘記（很適合套用）：
  - 開場立題：你在這趟旅行不會只待在一座城市 → 交通選擇必須「可複製」。
  - 先給一句總原則：車票通常越早訂越便宜（用一句話就好，後面再用數字補）。
  - 規則先拆成兩層：艙等（座位/服務）＋票價型（可改/可退/手續費）→ 讀者才知道你在比較什麼。
  - 用一次價差把道理釘住：同一套邏輯、不同購買時間/路線，價格會出現反直覺差距（用一組數字做例子就夠）。
  - 艙等推薦要有「焦慮解法」：不要只說更舒服；把它連到行李、出入口、驗票、候車動線（讀者會覺得你是在解決現場問題）。
  - 把候車寫成一個場景：貴賓室/候車區的價值不只是飲料，而是「等月台公布的混亂」被你隔在外面。
  - 寫出歐洲車站的節奏：月台資訊常接近發車才公布 → 人潮聚集 → 一公布就集體移動（這段張力很容易寫出來）。
  - 收尾回扣一句提醒：用「好險有買」這種回頭句，把整篇變成可行的建議，而不是炫耀服務。

- 一日攻略/村落一日（早餐→體驗→午餐→海景咖啡，這種節奏很好用）：
  - 用提問當開場：你喜歡怎樣去認識一座城町？（提問不是裝飾，它是在替「今天的走法」找理由。）
  - 先交代是「因緣」不是「計畫」：你是被帶去/剛好路過/臨時起心動念 → 才會有「意外發現魅力」的落差。
  - 地理定位一句就夠：距離大城市車程多久、在島/半島/山坡的哪一側（不要寫成百科）。
  - 用一個主題把一日釘住：陶器之鄉、咖啡小鎮、海風漁港…（主題一旦清楚，行程就會自己長出順序）。
  - 四段節奏最穩：早餐（人群/生活感）→ 上午（導覽/手作/在地體驗）→ 午餐（代表性料理）→ 下午（海景/高處/咖啡，把時間放慢）。
  - 在每段塞一個「畫面」：便當店清晨的忙碌、嘟嘟車的氣氛、手作時的手勢、海風與浪聲（每段 2–3 個就停）。
  - 資訊要用「小標題/短行」插入：店名、營業時間、集合點、官方帳號（像行程筆記，不要打斷敘事）。
  - 收尾用回訪願望：不是總結大道理，而是「我開始期待有一天能再來」這種乾淨句。

### 4) 子型 C：朝聖/延伸閱讀（影視/音樂/書）

- 不可思議開場：我竟然來了／我怎麼會在這裡。
- 允許落差：施工、人潮、現實（不要抱怨，承認即可）。
- 回補：一個救生圈把情緒補回來（遊客中心/明信片/路人互動）。
- 證明：讀完/看完後你做了什麼（把某地排進行程）。

### 5) 子型 D：旅遊誌長文（重返/同行/場景推進）

- 理由：為什麼回來（季節/同行/角度）。
- 推進：城市→荒野→秘徑→釋放視野（先壓住再打開）。
- 插曲：轉角小店/偶遇對話當作旅途禮物（只寫你真的遇到的）。
