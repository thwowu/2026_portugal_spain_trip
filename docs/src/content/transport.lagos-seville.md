# 拉各斯（Lagos）→ 塞維爾（Seville）

讀法：先看 `tldr` 抓結論；再看 `options`（bus/train）走細節。

## tldr
- recommended: bus
- because: 跨境用直達巴士最直覺；FlixBus 可直達塞維爾（Seville；常見到 Plaza de Armas 等站）。
- reminders:
  - ### Key takeaways
  - 先確認上車點與終點站（車票會寫清楚），避免到現場才找不到月台/站牌。
  - 再抓時間提早到站：一般 20–30 分鐘；旺季抓 30–45 分鐘。
  - 護照/證件隨身放（跨境可能備查），不要放托運行李。
  - 班次可能不多、時刻也比較固定：先買到票，再把 Seville 當天行程排上去。

## options
### bus | 巴士（直達、時間較短）
- summary: 直達省心；FlixBus 路線頁會列出 Seville 的主要下車點（Plaza de Armas / Prado / Santa Justa）。
- steps:
  - 先線上買票，確認「上車點」與「Seville 下車點」（會寫在車票上）。
  - 當天提早到 Lagos terminal → 找到月台/站牌 → 上車（如有托運就先處理行李）→ 抵達 Seville（依車票顯示站點）。
  - 到 Seville 後：先回住宿放行李、補水；跨境巴士有誤點的機率，行程別排太緊。
- bookingLinks:
  - FlixBus（Lagos → Seville 路線頁） | https://global.flixbus.com/bus-routes/bus-lagos-seville
- luggageNotes:
  - 直達較省事；貴重物品留隨身包
  - FlixBus：常見含 1 件托運行李 + 1 件手提行李（額外/特殊行李依官網規則） | https://global.flixbus.com/service/luggage
- riskNotes:
  - 誤點風險（塞車/跨境檢查/停靠站）。
  - 班次少：錯過通常就要改隔天，或改走「轉乘方案」。
- screenshots:
  - 巴士（示意） | /images/transport/lagos-seville-bus.png

### bus | 轉乘方案（經 法魯（Faro），更彈性但比較累）
- summary: 若直達班次不合/滿位：先到 法魯（Faro）（火車/巴士）→ 再接國際巴士到 塞維爾（Seville）。
- steps:
  - 第一步：Lagos → Faro（CP 火車或區域巴士，依你當天班次）。
  - 第二步：Faro → Seville（FlixBus 等國際巴士；依班次為準）。
  - 轉乘抓法：中間緩衝抓寬一點；最後一段別壓太晚，避免誤點後卡住。
- bookingLinks:
  - CP（班次/購票） | https://www.cp.pt/passageiros/en
  - FlixBus（法魯 Faro ↔ 塞維爾 Seville 路線入口） | https://global.flixbus.com/
- luggageNotes:
  - 多段轉乘需要搬行李；轉乘緩衝抓寬
- riskNotes:
  - 轉乘銜接風險較高；最後一段巴士別壓太晚
- screenshots:
