# 格拉納達延伸行程（詳情）

## alpujarras | Las Alpujarras（白村山景）
### summary
- 適合：好天氣的一日遊（山景/白村氛圍）
- 亮點：白色山村＋內華達山山麓風景

### images
- {{gallery:Las Alpujarras（白村山景）|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_en_Pampaneira._La_Alpujarra,_Granada..JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_Ver%C3%B3nica,_Pampaneira.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Alpujarras-Location1.PNG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_en_Pampaneira._La_Alpujarra,_Granada..JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_Ver%C3%B3nica,_Pampaneira.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Alpujarras-Location1.PNG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_en_Pampaneira._La_Alpujarra,_Granada..JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Calle_Ver%C3%B3nica,_Pampaneira.jpg?width=1800}}

### how
- 最穩做法：參加一日團/包車（山村路線轉乘麻煩、用錢換風險最低）
- 自駕：彈性最高（但山路彎多）

### route
- 典型一日團會跑數個白村（依團不同）：白村散步＋景觀點 + 午餐

### time
- 一日遊：通常 7–9 小時（含交通）

### backup
- 下雨/起霧：山景回報下降，改成 Guadix 或 Granada 市區（教堂/茶館/Hammam）

### sources
- Spain.info（Alpujarras）：https://www.spain.info/en/activities/discover-the-alpujarras-legado-andalusi-granada/
- Granada-Info（Alpujarra）：https://en.granada.info/alpujarra

## guadix | Guadix（洞穴城市）
### summary
- 適合：陰天或你想看「很不一樣的地景」
- 亮點：洞穴住居/地景感（半天到一天）

### images
- {{gallery:Guadix（洞穴住居）|https://commons.wikimedia.org/wiki/Special:FilePath/IAPH_Casas_cueva_Purullena.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/IAPH_Casas_cueva_Purullena.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/IAPH_Casas_cueva_Purullena.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/IAPH_Casas_cueva_Purullena.jpg?width=1800}}

### how
- 火車：Granada ↔ Guadix（可用 Renfe / Trainline 查班次）

### route
- 半日～一日：到 Guadix → 洞穴區/地景點 → 回 Granada

### time
- 單程約 1 小時上下（以班次為準）

### sources
- Visit Andalucia（Granada day trips / Guadix）：https://www.visit-andalucia.com/day-trips-from-granada/
- Trainline（Guadix ↔ Granada）：https://www.thetrainline.com/en/train-times/guadix-to-granada

## alhambra-ticketing | Alhambra 票務規則（備忘）
### summary
- 這不是行程，是避免踩雷：Nasrid Palaces 分時段，遲到通常就進不去

### images
- {{gallery:Alhambra（備忘）|https://commons.wikimedia.org/wiki/Special:FilePath/Alhambra_-_Granada.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Architectural_details_in_Alhambra,_Granada_(6930669668).jpg?width=1800|https://source.unsplash.com/gM1lnTTdWuE/1800x1200|https://source.unsplash.com/ryzLAmYJq1c/1800x1200|https://source.unsplash.com/gM1lnTTdWuE/1800x1200}}

### how
- 先做：上 Patronato 官方買票 → 選日期 + Nasrid Palaces 時段 → 把時段當作「全日錨點」
- 當天：至少提早 30–45 分鐘進園，因為園區走到入口要時間

### sources
- Patronato 官方票務：https://compratickets.alhambra-patronato.es/reservarEntradas.aspx?opc=142&gid=432&lg=en-GB&ca=0&m=GENERAL
- Patronato 入場時段規則：https://www.alhambra-patronato.es/en/visit/organize-your-visit/time-of-the-visit
