# {{bi:塞維爾|Seville}}→ {{bi:格拉納達|Granada}}

讀法：先看 `tldr` 抓結論；再看 `options`（bus/train）走細節。

## tldr
- recommended: train
- because: 整體更穩、也比較舒服；抵達後通常比較好接住宿休息。
- reminders:
  - ### Key takeaways
  - 先看你的目標：想舒服就火車；想省錢可改巴士（但更久）。
  - 帶行李就把「轉乘次數」降到最低，走起來會順很多。
  - 把 check-in 時間一起算：太早到可能要先寄放行李再回住宿，多跑一趟。

## options
### train | 火車（較舒適）
- summary: 約 2h40m～3h30m（班次依日期）。
- steps:
  - 先在 Renfe 查 `Sevilla` → `Granada`（站名常見：Sevilla-Santa Justa ↔ Granada）。
  - 線上買票後，當天提早到 Santa Justa → 找月台 → 上車 → 抵達 Granada。
  - 抵達後接市區：先查住宿離車站多遠；能搭公車就搭公車。行李日/晚到就別硬拖，短程計程車通常很便宜，直接搭，或把住宿選在「少走路、少轉乘」的位置。
- bookingLinks:
  - Renfe | https://www.renfe.com/es/en
- luggageNotes:
  - 行李上車放置較容易（比巴士好處理）
  - 抵達 Granada 後多半還要一段進城交通（先查住宿離車站距離）
- riskNotes:
  - 票價可能較高
  - 熱門時段需提前買
- screenshots:
  - 火車（示意） | /images/transport/seville-granada-train.png

### bus | 巴士（較便宜、較久）
- summary: 約 3 小時（示意）
- steps:
  - 先在線上買票，確認上車點與時間（常見：Seville Plaza de Armas）。
  - 當天提早到站 → 上車（如有托運就先處理行李）→ 抵達 Granada Station。
  - 抵達後：再用走路/公車/短程計程車接住宿（依你住宿位置選最省力的）。
- bookingLinks:
  - Alsa（Seville → Granada route） | https://www.alsa.com/en/coach/seville-granada
- luggageNotes:
  - 上車點固定，提早到站
  - 行李上下車搬運（貴重物品隨身）
- riskNotes:
  - 誤點風險
  - 旺季可能滿位
- screenshots:
  - 巴士（示意） | /images/transport/seville-granada-bus.png
