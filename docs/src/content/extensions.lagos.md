# 拉各斯（Lagos）延伸行程（詳情）

## sagres-cabo | 薩格雷什（Sagres）+ 聖文森特角（Cabo de São Vicente）（西岸懸崖/燈塔）

這是一個就算不能上船也會很有感的「西岸世界盡頭」日：強風、懸崖視野、燈塔的終點感會把你直接帶進大西洋尺度。懸崖邊風很大，最舒服的方式是短停看景、不要久站；交通用公車可行但班次務必抓寬，建議早出門、下午風更大就提早回。動線也別貪心，Sagres 或 Cabo 擇一就很值得——站在 Cabo de São Vicente 懸崖邊，你會很直覺理解那種「終點」或「出發」的感覺。

### images
{{gallery:Sagres + Cabo de São Vicente|https://commons.wikimedia.org/wiki/Special:FilePath/Sagres_-_Farol_de_Cabo_de_S%C3%A3o_Vicente.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cape_St._Vincent_Lighthouse.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Farol_Cabo_S%C3%A3o_Vicente_Sagres.jpg?width=1800|https://images.pexels.com/photos/24377407/pexels-photo-24377407.jpeg?auto=compress&cs=tinysrgb&w=1800}}

### when
最適合放：Benagil 取消/風浪大那天，或多住的加碼日（不需要好海象）。四月初風通常偏大，帶外套＋防風層。

### how
公車有路線但班次與時間要抓寬（以當季時刻為準）。

保守走法：只選 Sagres 或 Cabo 其中一個點就回（不要硬塞兩個點）。
一日動線：上午 Sagres（要塞/海岸）→ 下午 Cabo de São Vicente（燈塔/懸崖）→ 傍晚回 Lagos。一日遊：早出晚回（含交通）。

### backup
風太大：只留觀景點短停，別在懸崖邊久站。下雨：把戶外停留縮短，改成 Lagos 舊城區咖啡/餐廳休息。

## silves-monchique | 席爾維什（Silves）+ 蒙希克（Monchique）（古城 + 山區）

這是一個把阿爾加維從海岸線拉回「古城與地形」的選項：Silves 的紅色摩爾城堡與老城氛圍反差很強，陰天也不會太吃虧；若天氣好、想把視角拉到山上再加 Monchique（但視野很看天氣）。Silves 老城/城堡有坡、Monchique 更吃力；不自駕的話 Silves 火車/巴士可到但車站離老城有距離，而 Monchique 公車彈性低。保守做法就是只跑 Silves 半日～一日；想硬塞兩個點就得很早出門，否則交通會把整天吃掉。

### images
{{gallery:Silves + Monchique|https://commons.wikimedia.org/wiki/Special:FilePath/Silves_castle_-_ancient_capital_of_Algarve_-_The_Algarve,_Portugal_(1388844198).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Silves_castle_-_ancient_capital_of_Algarve_-_The_Algarve,_Portugal_(1387949771).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Monchique.JPG?width=1800}}

### how
去 Silves（不自駕也可）：火車 Lagos → Silves（車站離老城有距離）。巴士可能需要轉乘（常見在 Portimão 轉）。

去 Monchique：山區公車彈性較低（不自駕/不搭計程車就不建議硬塞）。
不自駕比較保守：Silves 半日（古城/城堡）→ 回 Portimão/Lagos（當作輕鬆日）。Silves 半日～一日（看你要不要加 Monchique）。

### backup
陰天/霧：Monchique 視野可能沒那麼值，改成 Silves + 早回海邊躺平。

## faro | 法魯（Faro）（舊城 + 補給日，一日）

這是一個舒服的「城市補給＋舊城慢走」日：在 Lagos 看了很多海岸線後，換到 Faro 的街區節奏會很像把旅行切回生活模式，順便把超市/藥妝/餐廳一次補齊。舊城散步為主、步行就能完成，火車/巴士都可達；不必拚很早，想更舒服就早去早回，午後留給喝咖啡與輕散步即可。

### images
{{gallery:Faro|https://commons.wikimedia.org/wiki/Special:FilePath/Faro_Old_Town.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Old_Town_-_Faro,_Portugal_(46471079874).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Faro_Largo_da_S%C3%A9_R01.jpg?width=1800}}

### how
火車/巴士：Lagos ↔ Faro（以當日班次為準；先查再排回程）。
上午到 Faro → 舊城散步/午餐 → 下午輕鬆走走 → 傍晚回 Lagos。建議抓整天或晚出早回（看你想多輕鬆）。

### backup
下雨：改成商場/咖啡＋短段散步（Faro 比海岸步道更好撤退）。

### sources
Wikivoyage（Faro）：https://en.wikivoyage.org/wiki/Faro

CP（查班次）：https://www.cp.pt/passageiros/en

## tavira | 塔維拉（Tavira）（東阿爾加維：更安靜的小城）

這是一個「不追大景、追舒服」的慢節奏小城：Tavira 的河岸與老城氛圍很生活，和 Lagos 的海岸線題材完全不同，適合你想把步調放慢的一天。走路量可控，但不自駕通常要先到 Faro 再轉乘，所以時間務必抓寬鬆、越早出發越不趕；如果你不想處理轉乘，就乾脆改去 Faro 做成更省腦的版本。

### images
{{gallery:Tavira|https://commons.wikimedia.org/wiki/Special:FilePath/Tavira_Portugal.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Tavira_(Portugal)_(12219403924).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Marina_de_Tavira_-_Portugal_(9400661120).jpg?width=1800}}

### how
交通通常需要先到 Faro 再接下一段（看當天班次；別把回程壓太晚）。不想轉乘就改去 Faro。
Lagos →（轉乘）→ Tavira → 老城/河岸慢走 → 回 Lagos。
建議抓整天（交通時間較長）。

### sources
Wikivoyage（Tavira）：https://en.wikivoyage.org/wiki/Tavira

## alvor | 阿爾沃爾（Alvor）（潟湖木棧道：最平最省體力）

這是最像「復健型半日」的選項：幾乎全平地的木棧道很適合膝蓋想休息但又想出門，或風浪太大不適合走岩岸的那天。它的價值不在景點密度，而在潟湖/濕地地形加上步道設計，讓你不用壓榨體力也能把海邊空氣與散步的舒服收好。公車可達但班次要看當天狀況，不搭計程車就別壓時間；半日就能完成，早去人少更順。

### images
{{gallery:Alvor|https://commons.wikimedia.org/wiki/Special:FilePath/Passadi%C3%A7os_de_Alvor_-_Portugal_(49381851036).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Rua_25_de_Abril,_Alvor,_18_September_2015_(3).JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Castelo_de_Alvor.JPG?width=1800}}

### how
交通以公車為主（看當天班次與你住宿位置）。
半日：Alvor 木棧道 → 咖啡/午餐 → 回 Lagos。半日很夠（也可做成輕鬆日）。

### sources
Wikivoyage（Alvor）：https://en.wikivoyage.org/wiki/Alvor

## carvoeiro-marinha | 卡爾沃埃羅（Carvoeiro）+ 馬里尼亞海灘（Praia da Marinha）（陸上岩岸代表作）

這是「不靠出海」也能看見阿爾加維代表級岩岸的陸上版本：Algar Seco 的天然窗/岩洞加上 Praia da Marinha 的觀景點，畫面感很直接。路線以觀景點＋短步道為主，路面可能不平，保守就只走最平的觀景點；不自駕需要抓公車與轉乘時間，不搭計程車就別把動線排太滿。建議早出發避免被轉乘卡死，累了就把行程縮到只做 Carvoeiro 也很有感——你看到的其實是「時間」把海岸雕刻成今天的樣子。

### images
{{gallery:Carvoeiro + Marinha|https://commons.wikimedia.org/wiki/Special:FilePath/Algar_Seco_rocks,_Carvoeiro,_Portugal_julesvernex2-4.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Praia_da_Marinha_-_Lagoa2.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Praia_da_Marinha_(35518737040).jpg?width=1800}}

### how
交通看公車與轉乘時間（別抓太緊；不搭計程車就更要保守）。
上午 Carvoeiro（Algar Seco）→ 午後 Marinha（觀景＋短步道）→ 回 Lagos。整天比較舒服（交通＋走路）。

### backup
只想半日：只做 Carvoeiro（Algar Seco）就很有感。
