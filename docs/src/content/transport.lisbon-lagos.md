# 里斯本（Lisbon）→ 拉各斯（Lagos）

🧳 先看 tldr｜🚌/🚆 再看 options

## tldr
- recommended: bus
- because: 直達、少轉乘；帶大行李最省心，且班次多（可選 Oriente / Sete Rios 出發）。
- reminders:
  - 出發站點先確認（常見：里斯本東方站 Oriente / 七河站 Sete Rios）
  - 提早到站 20–30 分鐘（旺季/連假抓 30–45 分）
  - 大行李：先在訂票時確認「可托運行李」與額外行李加購規則
  - 若想更舒適（但願意轉乘）：選 CP 火車（通常 Tunes 轉乘）

## options
### train | 火車（較舒適，但可能需要轉乘）
- summary: 舒適度高，但常見需要轉乘（通常 Tunes 轉乘；依班表可能更久）。
- steps:
  - 先在 CP 查 `Lisboa` → `Lagos`（很多時候會顯示需要在 Tunes 轉乘）
  - 若你不想賭轉乘：選「轉乘緩衝時間比較長」的班次
  - 旅遊當天：提早到站 → 找月台 → 上車 → 圖內什（Tunes）轉乘 → 抵達 拉各斯（Lagos）後再用走路/市內公車接住宿（若行李很重/晚到：短程計程車通常很便宜，直接搭；或事前把住宿選在車站/terminal 附近）
- bookingLinks:
  - CP（班次/購票） | https://www.cp.pt/passageiros/en
  - CP（線上購票入口） | https://www.cp.pt/passageiros/en/buy-tickets
- luggageNotes:
  - 轉乘時需要搬行李（樓梯/電梯狀況看車站）
  - 行李多：優先選轉乘時間更寬鬆的班次，降低壓力
- riskNotes:
  - 轉乘時間太緊會增加錯過風險
  - 班表/票價會隨日期變動（以 CP 當下顯示為準）
- screenshots:

### bus | 巴士（直達、最省事）
- summary: 直達最省心；常見到 Lagos bus terminal，適合帶行李與不想轉乘的人。
- steps:
  - 先選出發站：里斯本東方站（Oriente）或七河站（Sete Rios）（以你住宿位置/地鐵轉乘最順者為準）
  - 線上買票 → 當天提早到站 → 找到月台/站牌 → 上車行李托運（如適用）→ 抵達 Lagos terminal
  - 抵達後：走路到住宿（Old Town/Marina 一帶通常很近；若行李很重/晚到：短程計程車通常很便宜，直接搭；或把住宿選在 terminal 附近）
- bookingLinks:
  - FlixBus（Lisbon → Lagos 路線頁） | https://global.flixbus.com/bus-routes/bus-lisbon-lagos
  - Rede Expressos（官網/APP） | https://rede-expressos.pt/en
- luggageNotes:
  - 直達不用轉乘搬運
  - FlixBus：常見含 1 件托運行李 + 1 件手提行李（額外/特殊行李要加購，依官網規則） | https://global.flixbus.com/service/luggage
  - 上車前把行李標記好；貴重物品留在隨身包
- riskNotes:
  - 主要風險是塞車/誤點（出發前用官方 tracker 看車況）
  - 出發站點（Oriente/Sete Rios）要先確認，避免跑錯站
