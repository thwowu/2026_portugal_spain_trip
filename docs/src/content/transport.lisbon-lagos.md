# {{bi:里斯本|Lisbon}}→ {{bi:拉各斯|Lagos}}

讀法：先看 `tldr` 抓結論；再看 `options`（bus/train）走細節。

## tldr
- recommended: bus
- because: 直達、少轉乘；帶大行李比較不折騰，班次也多（常見從 Oriente / Sete Rios 出發）。
- reminders:
  - ### Key takeaways
  - 先確認出發站點（常見：里斯本東方站 Oriente / 七河站 Sete Rios），避免跑錯站。
  - 再抓時間提早到站：一般 20–30 分鐘；旺季/連假抓 30–45 分鐘。
  - 有大行李：訂票時先確認「可托運行李」與額外行李加購規則；貴重物品留在隨身包。
  - 想坐得更舒服、也能接受轉乘：選 CP 火車（很多班次會在 Tunes 轉乘）。

## options
### train | 火車（較舒適，但可能需要轉乘）
- summary: 舒適度高，但常見需要轉乘（通常 Tunes 轉乘；依班表可能更久）。
- steps:
  - 先在 CP 查 `Lisboa` → `Lagos`（很多班次會顯示需要在 Tunes 轉乘）。
  - 不想趕轉乘：選「轉乘緩衝時間比較長」的班次，壓力會小很多。
  - 旅遊當天照這個順序走：提早到站 → 找月台 → 上車 → 到 Tunes 轉乘 → 抵達 Lagos。
  - 抵達後接住宿：能走就走；行李很重/晚到就搭短程計程車（通常不貴），或事前把住宿選在車站/terminal 附近。
- bookingLinks:
  - CP（班次/購票） | https://www.cp.pt/passageiros/en
  - CP（線上購票入口） | https://www.cp.pt/passageiros/en/buy-tickets
- luggageNotes:
  - 轉乘時需要搬行李（樓梯/電梯狀況看車站）。
  - 行李多：優先選轉乘時間更寬鬆的班次，降低壓力。
- riskNotes:
  - 轉乘時間太緊會增加錯過風險。
  - 班表/票價會隨日期變動（以 CP 當下顯示為準）。

### bus | 巴士（直達、最省事）
- summary: 直達最省心；常見到 Lagos bus terminal，適合帶行李與不想轉乘的人。
- steps:
  - 先選出發站：Oriente 或 Sete Rios（以你住宿位置/地鐵轉乘最順者為準）。
  - 線上買票後，當天提早到站 → 找到月台/站牌 → 上車（如有托運就先處理行李）→ 抵達 Lagos terminal。
  - 抵達後接住宿：Old Town/Marina 一帶多半能走到；行李很重/晚到就搭短程計程車，或事前把住宿選在 terminal 附近。
- bookingLinks:
  - FlixBus（Lisbon → Lagos 路線頁） | https://global.flixbus.com/bus-routes/bus-lisbon-lagos
  - Rede Expressos（官網/APP） | https://rede-expressos.pt/en
- luggageNotes:
  - 直達不用轉乘搬運
  - FlixBus：常見含 1 件托運行李 + 1 件手提行李（額外/特殊行李要加購，依官網規則） | https://global.flixbus.com/service/luggage
  - 上車前把行李標記好；貴重物品留在隨身包
- riskNotes:
  - 主要風險是塞車/誤點（出發前可用官方 tracker 看車況）。
  - {{bi:出發站點|Oriente/Sete Rios}}要先確認，避免跑錯站。
