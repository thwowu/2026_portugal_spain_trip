# 塞維爾（Seville）延伸行程（詳情）

閱讀方式：
- 先看每段最上面 3 行：**地形（樓梯/上坡）**、**交通（能否全程大眾運輸）**、**早起安排（早班人少）**
- 如果只想選 1 個：**Córdoba（風景＋老城密度高）**；想做半日就選 **Itálica（戶外遺址）**

## cordoba | 科爾多瓦（Córdoba；一日：清真寺主教座堂 + 迷宮老城）
- 🧗 地形：老城以步行為主（巷弄好走，但走久會累；保守就縮短動線）
- 🚆 交通：火車 + 走路/市區公車；如果你們想省體力，就把行程做成「老城小圈慢走」版本（少走路、少繞路）；真的卡在「車站到老城這段」就直接短程計程車（通常很便宜）
- ⏰ 早起：搭早班到，先做河岸/橋視角，再進 Mezquita（人少比較舒服）

### summary
- 適合：你想要「一日就很有收穫」的延伸；或 Semana Santa / 大週封路時想換到步行更舒服的城市
- 亮點：Mezquita-Catedral（清真寺主教座堂）本體超震撼、老城集中好走、拍照回報高

### why
- Córdoba 的爽點是「密度高、回報大」：一日不用拉很遠也不用一直換交通工具，就能在 Mezquita-Catedral 這種把清真寺柱林與天主教中殿疊在同一空間的奇觀裡，直接感受政權更迭留下的拼貼（伊斯蘭統治時期曾是歐洲最重要城市之一 → 基督教收復後改造並「嵌入」主教座堂），走路就能收滿老城。

### how
- 火車：Renfe（Sevilla-Santa Justa ↔ Córdoba；常見約 45 分鐘級，實際以班次為準）
- 進城（最後一哩）：Córdoba 車站到老城「走路約 25–35 分鐘」；不想走太多就改搭市區公車（以路況/當地路線為準）
- 建議：先用 Renfe 鎖「去/回程班次」，再去鎖 Mezquita 的票（旺季/週末更重要）

### route
- 必做清單：Mezquita-Catedral →（走路順）猶太區巷弄（Judería）→ 羅馬橋（Puente Romano）→ 河岸夕陽（時間夠再加 Alcázar）
- 版本 A（經典不趕，推薦）：早班車到 Córdoba → Mezquita（先衝）→ 猶太區巷弄/午餐 → 羅馬橋/河岸 → 下午回 Sevilla
- 版本 B（下雨友善）：Mezquita（室內）→（選配）Alcázar/博物館 → 早點回 Sevilla
- 版本 C（只想輕鬆）：Mezquita + 老城慢走 + 咖啡，其他全放掉也成立

### time
- 抓法：往返交通 + 老城步行為主，建議整天（早出晚回最舒服）

### cost
- Mezquita：建議直接看官方票務/時段（旺季「現場排隊」風險高）
- 小提醒：有些時段人潮會集中（團客），想舒服就選更早或更晚的入場（視季節）

### backup
- Sevilla 市中心太擠/封路：Córdoba 是很強的「逃離人潮」選項，但本身也可能人多，務必早出門

## cadiz | 加的斯（Cádiz；老城 + 海邊：一日「放空回血」）
- 🧗 地形：老城多平地，最適合「慢慢走、走到哪算哪」
- 🚆 交通：火車到站後主要靠走路（不搭計程車也很順）
- ⏰ 早起：早到海邊人少；也可中午出發做成保守半日

### summary
- 適合：你已經看膩宮殿教堂了，想要一個「不費腦、但很舒服」的看海日
- 亮點：老城散步（不趕行程也很有感）+ La Caleta 海邊日落（拍照/放空都行）

### why
- Cádiz 是把題材從宮殿教堂一鍵切到「海風與慢步調」的回血日：老城幾乎被海包住、你不用刻意找景點光是走就會有畫面（配 La Caleta 日落很有效），而它之所以這麼“海”，是因為自腓尼基人以來就被當作海上貿易據點，後續羅馬與近代海權也讓城市長期與港口綁在一起。

### images
- {{gallery:Cádiz|https://commons.wikimedia.org/wiki/Special:FilePath/Playa_de_la_Caleta,_C%C3%A1diz,_Espa%C3%B1a,_2015-12-08,_DD_53.JPG?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz_playa_caleta.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Cadiz._Puesta_de_sol_en_Mayo.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Exterior_Catedral_de_C%C3%A1diz_1.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Kathedrale_in_C%C3%A1diz.jpg?width=1800}}

### how
- 火車：Sevilla-Santa Justa ↔ Cádiz（常見約 1h40–1h50 級；以當天班次為準）
- 票務策略：先鎖回程（避免你在海邊太爽忘記時間，最後被迫等很久）

### route
- 這天的關鍵：不要塞滿景點，把節奏放慢（走走、吃海鮮、吹海風就夠）
- 版本 A（最舒服的經典）：到 Cádiz → 老城慢走（不趕）→ 市場/海鮮午餐 → 午後 La Caleta + 海岸線 → 日落 → 回 Sevilla
- 版本 B（雨天也能活）：老城 + 市場 + 咖啡店，海邊只做短停（風雨大就直接放棄海岸線）
- 版本 C（你只想半日）：中午出發 → La Caleta + 老城吃晚餐 → 晚班回 Sevilla

### time
- 單程約 1h40 左右（以班次為準）；市內主要靠走路（很適合當「走路復健日」）

### cost
- 預算抓法：市場先吃一輪（省踩雷）→ 海邊再補一杯咖啡/甜點（把它當作「旅行中場休息」）

## italica | Itálica（羅馬遺址半日：用最小成本換最大「哇」）
- 🧗 地形：戶外為主、空曠好走（帶水/防曬；保守就只走精華）
- 🚆 交通：公車可達（符合「不搭計程車」）
- ⏰ 早起：上午去最舒服（太陽沒那麼大、人也少）

### summary
- 適合：你想加一個「半日就收工」的高 CP 行程；或 Sevilla 市中心太擠、想去空曠地方走走
- 亮點：圓形劇場 + 馬賽克遺址，畫面感很強，而且幾乎不用排隊

### why
- Itálica 用半日就能把 Sevilla 的「近代王權/宗教城市」往前拉到「羅馬日常」的時間尺度：圓形劇場、馬賽克、街廓畫面感很強又不太需要搶票，順便讓你理解羅馬帝國如何把城市生活「複製」到伊比利半島（它也常被提到與 Trajan、Hadrian 等皇帝的淵源），非常適合當緩衝日。

### images
- {{gallery:Itálica|https://commons.wikimedia.org/wiki/Special:FilePath/Italica_Amphitheatre_Seville.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Anfiteatro_de_It%C3%A1lica_(Santiponce,_Sevilla).jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Italica_-_Mosaico.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Italica_-_Mosaics.jpg?width=1800}}

### how
- 公車：Sevilla → Santiponce（Itálica 所在地；常見從 Plaza de Armas 一帶出發，路線/班次以當季公告為準）
- 裝備：防曬 + 水；春天也可能風大，薄外套很加分

### route
- 必看順序：先圓形劇場（Anfiteatro）→ 再走一圈馬賽克遺址（不用追全走完，挑你最有感的幾個點就好）
- 版本 A（半日精華，推薦）：早上出發 → 先圓形劇場 → 再走一圈馬賽克遺址 → 中午回 Sevilla 吃飯/午休
- 版本 B（你只剩 2 小時）：只做圓形劇場 + 最近的遺址區，拍夠就撤（不要硬走全區）

### time
- 交通 + 參觀：抓 4–6 小時最舒服；想更輕鬆就抓「上午半天」即可

### cost
- 開放時間/票務會隨季節變動：出發前用官方或現場公告確認（別把最後入場抓太緊）
 - 小提醒：週一常見休館，別把行程壓到最後一天（仍以官方公告為準）

### backup
- 太熱/太累：直接砍掉「遺址街區」，留圓形劇場就好（回報仍然很高）

## ronda | Ronda（白色山城/峽谷橋：把「明信片」走進現實，一日）
- 🧗 地形：要走的路比較多（保守就只看新橋＋兩個視角點）
- 🚆 交通：多數情況巴士最直覺（先查去回程，別壓太晚）
- ⏰ 早起：越早出發越不趕（拍照停留時間會被拉長）

### summary
- 適合：你想看安達魯西亞最經典的峽谷城（而且拍照回報爆表）
- 亮點：Puente Nuevo 橫跨深谷的那一瞬間，會讓你覺得「有來值得」

### why
- Ronda 的核心是垂直地形震撼：房子、橋、峽谷直接疊在一起，當作安達魯西亞白村/山城代表作一個點就能收滿氛圍，而它的故事其實是「地形把城切成兩半 → 人用橋回答地形」，Puente Nuevo 重新把城市接回來也因此成為大家記得 Ronda 的原因。

### images
- {{gallery:Ronda|https://commons.wikimedia.org/wiki/Special:FilePath/Puente_Nuevo,_Ronda,_Andaluc%C3%ADa,_Spain.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Ronda_Puente_Nuevo.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Ronda_02.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Puente_Nuevo,_Ronda,_Andaluc%C3%ADa,_Spain.jpg?width=1800}}

### how
- 交通：多數情況「巴士比火車更直覺」（火車常繞路/耗時）；先查當天班次再排午餐與回程
- 票務：旺季/週末提早買，避免臨時沒座位（尤其回 Sevilla 的班次）

### route
- 必看順序：新城（Mercadillo）→ Puente Nuevo（新橋）+ 觀景 → 老城（La Ciudad）巷弄慢走（把它當散步，不要當闖關）
- 版本 A（經典不趕，推薦）：早出發 → 抵達後先走到新橋 + 觀景點 → 午餐 → 老城慢走 → 傍晚回 Sevilla
- 版本 B（你怕走太累）：只做新橋 + 兩個視角點 + 咖啡/午餐，老城只鑽一小段巷弄就撤

### time
- 建議抓整天（含往返交通）；Ronda 本體步行時間容易被「拍照停留」拉長，別把回程壓太緊

### backup
- 若你只剩半天：Ronda 不建議硬塞（交通成本太高），改去 Itálica 或 Carmona（半日就能收）

### sources

- Rome2Rio（Seville ↔ Ronda 交通彙整）：https://www.rome2rio.com/Bus/Seville/Ronda

## jerez | Jerez de la Frontera（雪莉酒/馬術：主題型一日）
- 🧗 地形：主題型行程，可走得很輕鬆（不需要一直走景點）
- 🚆 交通：火車可達；先選「酒莊」或「馬術」場次，再配火車
- ⏰ 早起：想要更保守就只做其中一個主題（別兩個都塞）

### summary
- 適合：你想把一天換成「安達魯西亞三件套：酒、馬、flamenco 氛圍」的一日，而不是再看第 N 座教堂
- 亮點：Sherry bodegas（導覽/品飲）+ 皇家馬術學校表演（選一個就很夠，兩個都做也成立）

### why
- Jerez 比起再看第 N 座教堂，更像把一天換成「酒、馬、flamenco 氛圍」的生活文化課：雪莉酒酒莊與馬術技藝把城市的日常做成儀式感，尤其當你已經看了很多古蹟時很能換腦，而它之所以成為 Jerez 的名片，正是 Sherry 的產區/貿易把它推上國際、加上安達魯西亞馬術傳統在此被制度化（學校、表演、訓練）。

### images
- {{gallery:Jerez|https://commons.wikimedia.org/wiki/Special:FilePath/Jerez_de_la_Frontera_-_Alc%C3%A1zar.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Jerez_de_la_Frontera_-_Catedral.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Jardines_Alcazar_de_Jerez_de_la_Frontera.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Mezquita,_Alc%C3%A1zar,_Jerez_de_la_Frontera,_Espa%C3%B1a,_2015-12-07,_DD_57.JPG?width=1800}}

### how
- 火車：Sevilla-Santa Justa ↔ Jerez de la Frontera（先查班次，再配合你想去的酒莊/表演場次）
- 票務策略：先鎖「酒莊或表演」的場次（這是時間錨點）→ 再鎖火車

### route
- 三選一就很夠：① 1 間 bodegas 導覽/品飲 ② 馬術表演 ③ Alcázar + 市中心散步（兩個以上就會很滿）
- 版本 A（酒莊 + 城市散步）：早到 → bodegas 導覽 → 午餐 → Alcázar/市中心 → 回 Sevilla
- 版本 B（馬術優先）：按表演時間排 → 表演前後插市中心散步/午餐 → 回 Sevilla

### time
- 抓整天最穩；若你只做「酒莊」或「馬術」單一主題，也可做成「晚出早回」

### backup
- 若你不喝酒/不看表演：把 Jerez 改成「Cádiz 看海」或「Itálica 半日」會更直覺

## carmona | Carmona（近郊古城半日：最像「白色小鎮」但超省時）
- 🧗 地形：老城巷弄慢走（有些坡，但半日就能收）
- 🚆 交通：巴士為主（先查好去回程最保守）
- ⏰ 早起：早到拍完、吃完午餐就回 Sevilla（下午留給休息）

### summary
- 適合：你想要「白色小鎮感」但不想跑太遠（比 Ronda 更省時省力）
- 亮點：城門、城牆、巷弄與高點視野；半日就很有感

### why
- Carmona 像把安達魯西亞小鎮精華濃縮成「白牆巷弄＋防禦古城感＋高處視野」：不用跑遠、不用搶票也不用早起到崩潰就能換到完全不同的氛圍，很適合當行程緩衝，而它長期被各時期政權爭著掌握的原因也很直覺——地勢高、天然好守，你看到的城門/城牆就是地形變成權力的痕跡。

### images
- {{gallery:Carmona|https://commons.wikimedia.org/wiki/Special:FilePath/Carmona_-_Puerta_de_Sevilla.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Alc%C3%A1zar_Puerta_Sevilla_Carmona.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Carmona_-_Spain.jpg?width=1800|https://commons.wikimedia.org/wiki/Special:FilePath/Carmona_-_Andalousie_mai_2018_103_(40427868293).jpg?width=1800}}

### how
- 交通：巴士為主（班次可能不密，先查好去/回程再排午餐）

### route
- 必走清單：Puerta de Sevilla（城門）→ 老城巷弄慢走 → 找一個高點/觀景（選一個最舒服的就好）
- 半日（最順）：早上到 Carmona → 城門/老城巷弄慢走 → 午餐/咖啡 → 回 Sevilla 下午繼續逛市區

### time
- 交通 + 參觀：抓 4–6 小時最舒服

### backup
- 若你當天只想休息：直接留在 Sevilla 的 Triana/河岸散步也完全成立

## aracena | Aracena（洞穴 + 火腿：山區換腦一日）
- 🧗 地形：山區小鎮（可能有坡；保守就把步行縮短）
- 🚆 交通：巴士往返（距離較遠，要先看時刻；不搭計程車就更要抓寬鬆）
- ⏰ 早起：建議非常早出發、早點回（不然會累到晚上）

### summary
- 適合：你想看「山區小鎮＋洞穴」的特殊題材，順便認真吃一次火腿
- 亮點：Gruta de las Maravillas（洞穴）+ 山城氛圍（跟 Sevilla 完全不同）

### why
- Aracena 是題材跳躍最有效的一天：從安達魯西亞大城的石板路切到山裡洞穴與火腿產地，記憶點會突然變立體（而且很在地），背後原因也很“地理”——Sierra de Aracena 的山地環境孕育傳統畜牧/火腿文化，洞穴則用「地球的時間」把你從城市的時間感拉開。

### how
- 巴士：Sevilla（常見 Plaza de Armas）↔ Aracena（以當季時刻為準）
- 票務策略：先看洞穴導覽的場次/能否預約（這是時間錨點）→ 再配巴士班次

### route
- 走法重點：先洞穴（避免下午票賣光）→ 再慢慢吃午餐/火腿 → 下午散步/高點（不要把自己趕爆）
- 一日（不趕版）：早出發 → 先洞穴（避免下午票賣光）→ 小鎮午餐/火腿 → 下午慢走/高點 → 回 Sevilla

### time
- 建議抓整天（交通時間較長）；如果你只剩半天不要硬塞

### backup
- 若你不想跑遠：改成 Itálica 或 Carmona（半日解決，回報也高）
