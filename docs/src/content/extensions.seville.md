# 塞維爾（Seville）延伸行程（詳情）

## cordoba | 科爾多瓦（Córdoba；一日：清真寺主教座堂 + 迷宮老城）

這是一個一日就很有收穫的延伸選項，也很適合 Semana Santa 封路擁擠時「逃離人潮」到一個步行更舒服的城市：Mezquita-Catedral 的柱林與中殿拼貼感非常震撼，老城密度高、拍照回報也高。地形上以步行為主（巷弄好走但走久會累），交通用火車＋走路/市區公車即可成立；想省力就把動線做成「老城小圈慢走」版本，卡在車站到老城這段再用短程計程車也行。建議搭早班到，先做河岸/橋視角再進 Mezquita，人少會舒服很多。

### how
火車：Renfe（Sevilla-Santa Justa ↔ Córdoba；常見約 45 分鐘級，實際以班次為準）。

最後一哩：Córdoba 車站到老城走路約 25–35 分鐘。不想走太多就改搭市區公車（以路況/當地路線為準）。

建議：先用 Renfe 鎖去/回程班次，再去鎖 Mezquita 的票（旺季/週末更重要）。
必做清單：Mezquita-Catedral →（走路順）猶太區巷弄（Judería）→ 羅馬橋（Puente Romano）→ 河岸（時間夠再加 Alcázar）。

版本 A（經典不趕）：早班車到 Córdoba → Mezquita（先衝）→ 猶太區巷弄/午餐 → 羅馬橋/河岸 → 下午回 Sevilla。

版本 B（下雨友善）：Mezquita（室內）→（選配）Alcázar/博物館 → 早點回 Sevilla。

版本 C（只想輕鬆）：Mezquita + 老城慢走 + 咖啡，其他全放掉也成立。
往返交通 + 老城步行為主，建議整天（早出晚回最舒服）。

### cost
Mezquita 建議直接看官方票務/時段（旺季現場排隊風險高）。想舒服就選更早或更晚入場（視季節）。

### backup
Sevilla 市中心太擠/封路：Córdoba 是很強的逃離人潮選項，但本身也可能人多，務必早出門。

## cadiz | 加的斯（Cádiz；老城 + 海邊：一日放空回血）

如果你已經看膩宮殿教堂，這是一個不費腦但很舒服的看海回血日：老城幾乎被海包住，你不用刻意找景點光是走就會有畫面，最後再用 La Caleta 日落收尾（拍照或放空都行）。老城多平地很適合慢慢走、走到哪算哪；火車到站後主要靠走路，不搭計程車也很順。早到海邊人會少很多，但也能中午出發做成保守半日——重點是把節奏放慢，別把行程塞滿。

### how
火車：Sevilla-Santa Justa ↔ Cádiz（常見約 1h40–1h50 級；以當天班次為準）。

票務策略：先鎖回程（避免在海邊太爽忘記時間，最後被迫等很久）。
關鍵：不要塞滿景點，把節奏放慢（走走、吃海鮮、吹海風就夠）。

版本 A（經典）：到 Cádiz → 老城慢走 → 市場/海鮮午餐 → 午後 La Caleta + 海岸線 → 日落 → 回 Sevilla。

版本 B（雨天也能活）：老城 + 市場 + 咖啡店，海邊只做短停（風雨大就直接放棄海岸線）。

版本 C（只想半日）：中午出發 → La Caleta + 老城吃晚餐 → 晚班回 Sevilla。
單程約 1h40 左右（以班次為準）；市內主要靠走路（很適合當走路復健日）。

### cost
預算抓法：市場先吃一輪（省踩雷）→ 海邊再補一杯咖啡/甜點（當作旅行中場休息）。

## italica | Itálica（羅馬遺址半日：用最小成本換最大哇）

這是一個半日就能收工的高 CP 選項，也很適合 Sevilla 市中心太擠時去空曠地方走走：圓形劇場＋馬賽克遺址畫面感很強，通常也不太需要排隊。戶外為主、空曠好走，但要帶水與防曬（保守就只走精華）；交通公車可達，符合不搭計程車的版本。上午去最舒服（太陽沒那麼大、人也少），也能用這半天把時間尺度從 Sevilla 的近代王權/宗教城市，往前拉回羅馬日常，當作很好的緩衝日。

### how
公車：Sevilla → Santiponce（Itálica 所在地；常見從 Plaza de Armas 一帶出發，路線/班次以當季公告為準）。

裝備：防曬 + 水；春天也可能風大，薄外套很加分。
必看順序：先圓形劇場（Anfiteatro）→ 再走一圈馬賽克遺址（不用追全走完，挑你最有感的幾個點就好）。

版本 A（半日精華）：早上出發 → 先圓形劇場 → 再走一圈遺址 → 中午回 Sevilla 吃飯/午休。

版本 B（只剩 2 小時）：只做圓形劇場 + 最近的遺址區，拍夠就撤。
交通 + 參觀：抓 4–6 小時最舒服；想更輕鬆就抓上午半天即可。

### backup
太熱/太累：直接砍掉遺址街區，留圓形劇場就好（回報仍然很高）。
